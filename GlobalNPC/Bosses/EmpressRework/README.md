# Empress of Light - Rework Boss Mod

## Overview
Mod yang merework Empress of Light menjadi boss yang sangat sulit dengan style Infernum Fargo.

## Features

### 🎮 Gameplay
- **3 Phase System**: Phase 1 (100-66%), Phase 2 (65-34%), Phase 3/DSP (<33%)
- **5-7 Attack Patterns per Phase**: Total 21 attack variations
- **Telegraphed Attacks**: Visual warning sebelum serangan dimulai
- **Progressive Difficulty**: Setiap phase lebih sulit dari sebelumnya

### ⚔️ Phase Details

#### Phase 1 (100-66% HP) - Introduction
1. Homing Burst Attack - Proyektil memutar mengelilingi boss
2. Laser Beam Pattern - Beam lasers tertarget
3. Charge Attack - Boss menerjang ke pemain
4. Projectile Circle - Lingkaran proyektil 360°
5. Wave Pattern - Gelombang proyektil beruntun
6. Spin Attack - Rotasi 16 proyektil
7. Targeted Spray - Proyektil acak tertarget

**Cooldown**: 90 frames (1.5 detik)
**Telegraph Delay**: 30 frames (0.5 detik)

#### Phase 2 (65-34% HP) - Intensified
1. Combo Attack - Burst ganda dengan 10 proyektil
2. Screen Filling Projectiles - Proyektil spam 20 arah
3. Dash Attack - Boss mengejar + proyektil
4. Cross Laser - Pola silang proyektil
5. Spiral Pattern - Spiral dengan proyektil bertambah
6. Double Wave - Dua gelombang proyektil
7. Chaotic Burst - Acak menyeluruh 25 proyektil

**Cooldown**: 75 frames (1.25 detik)
**Telegraph Delay**: 25 frames (0.42 detik)

#### Phase 3 / DSP (<33% HP) - Extreme
1. Extreme Combo - Triple burst dengan rotasi
2. Projectile Hell - 30 proyektil spam maksimal
3. Multi Dash - 3 arah dash + proyektil
4. Laser Grid - Grid 5x5 proyektil
5. Chaos Pattern - 35 proyektil random
6. Ultimate Burst - 40 proyektil maximum
7. Inferno Phase - Infernum-style beam attack

**Cooldown**: 60 frames (1 detik)
**Telegraph Delay**: 20 frames (0.33 detik)

### 🎨 Visual Effects
- Telegraph glow effect sebelum attack
- Dust particles saat attack
- Color-coded visual warnings
- Smooth particle transitions

### 📊 Stats

**Health**: 4200 (original: 3600) +600 HP
**Damage**: 55 (original: 50) +5 damage
**Defense**: 15 (original: 10) +5 defense

## Installation

1. Copy folder ke `Mods/TheSanity/GlobalNPC/Bosses/`
2. Reload mods di tModLoader
3. Summon Empress dengan Crystal Heart

## Difficulty Balance

Mod ini dirancang untuk:
- Pemain yang menginginkan challenge extreme
- Style Infernum Fargo dengan visual menarik
- Attack pattern kompleks yang memerlukan skill
- Telegraphed attacks untuk fair gameplay

## Notes

- Projectile menggunakan FairyQueenMagicProjectile untuk visual konsisten
- Sound effects menggunakan default Terraria
- Sprite asli Empress digunakan
- Balanced untuk multiplayer dan single-player
